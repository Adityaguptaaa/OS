Frequently Asked Questions

How do I start a simulation?

Navigate to the Dashboard, create a new simulation, add processes, configure IPC channels, and click "Start Simulation".

What IPC types are supported?

The debugger supports three IPC mechanisms: Pipes (unidirectional), Message Queues (priority-based), and Shared Memory (fastest).

How does deadlock detection work?

The system uses Resource Allocation Graph (RAG) analysis with cycle detection algorithms to identify circular wait conditions between processes.

Can I export simulation data?

Yes! Visit the Logs page to export event logs in JSON or CSV format for further analysis.

How do I run this locally?

Install Python dependencies with `pip install -r requirements.txt`, then run `python run.py`. Access the application at http://localhost:5000.
